Community Healthcare Management System

A full-stack healthcare management system built to reflect real-world clinic operations through clean architecture and API-driven design.

. Overview

This project separates core entities such as clinics, doctors, patients, and rooms, and manages their relationships dynamically instead of relying on rigid, destructive CRUD operations.

The result is a flexible and realistic healthcare management model.
